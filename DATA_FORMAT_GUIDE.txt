===============================================
BULK LOAN APPLICATION - REQUIRED DATA FORMAT
===============================================

FORMAT: JSONL (One JSON object per line)

REQUIRED FIELDS MAPPING:
========================

YOUR DATA                           →  REQUIRED FIELD NAME
-----------                            -------------------

personal_info.name                  →  applicant.firstName + applicant.lastName (split the name)
personal_info.age                   →  Calculate dob from age
personal_info.gender                →  applicant.gender
personal_info.marital_status        →  maritalStatus (values: "Single", "Married", "Divorced", "Widowed")
personal_info.dependents            →  dependents array (optional, can be empty [])

contact_info.phone                  →  applicant.phone (10 digits only, remove extensions)
contact_info.email                  →  applicant.email
contact_info.address                →  applicant.address + applicant.city + applicant.state (split address)

loan_details.loan_amount            →  loanAmount
loan_details.loan_term_months       →  tenureMonths
loan_details.loan_type              →  loanType (values: "home", "gold", "personal", "vehicle", "education", "business")
loan_details.interest_rate          →  (Not in DTO - will be ignored)
loan_details.collateral             →  collaterals array (optional)

employment_info.employment_type     →  employmentType (values: "salaried", "self-employed")
employment_info.annual_income       →  monthlyIncome (divide by 12)
employment_info.employer            →  employerName
employment_info.designation         →  designation
employment_info.years_with_employer →  Calculate employmentStartDate

document_info.aadhar_number         →  aadhaarNumber (12 digits)
document_info.pan_number            →  panNumber (format: ABCDE1234F - 5 letters, 4 digits, 1 letter)
document_info.passport_number       →  passportNumber
document_info.driving_license       →  voterId (or create voterId)
document_info.bank_account_number   →  accountNumber
document_info.ifsc_code             →  ifscCode (format: ABCD0123456)

financial_history.credit_score      →  creditScore (300-900)
financial_history.previous_loans    →  totalActiveLoans (count)
financial_history.previous_default  →  defaultsCount (0 if false, >0 if true)
financial_history.existing_emi      →  totalMonthlyEmi

ADDITIONAL REQUIRED FIELDS:
===========================
- education: String (e.g., "Bachelor of Science", "Master of Business Administration")
- nationality: String (default: "Indian")
- bankName: String (e.g., "HDFC Bank", "ICICI Bank", "State Bank of India")
- accountType: String (values: "savings", "current")
- totalCreditLastMonth: BigDecimal
- totalDebitLastMonth: BigDecimal
- anomalies: String (default: "No anomalies detected")
- residenceType: String (values: "owned", "rented", "parental", "company_provided")
- propertyOwnership: String
- monthlyRent: BigDecimal (0 if owned)
- yearsAtCurrentAddress: Integer
- propertyValue: BigDecimal
- propertyType: String (values: "apartment", "independent_house", "villa", "farmhouse")
- totalAreaSqft: Integer
- hasHomeLoan: Boolean
- outstandingHomeLoan: BigDecimal
- homeLoanEmi: BigDecimal
- creditBureau: String (values: "CIBIL", "Experian", "Equifax", "CRIF")
- totalOutstandingDebt: BigDecimal
- totalCreditLimit: BigDecimal
- creditUtilizationRatio: BigDecimal (0.0 to 1.0)
- paymentHistory: String (values: "excellent", "good", "fair", "poor")
- bankruptcyFiled: Boolean
- creditReportUrl: String (optional)
- documents: Array (can be empty [])
- references: Array (can be empty [])
- coApplicants: Array (can be empty [])
- collaterals: Array (can be empty [])

VALIDATION RULES:
=================
- panNumber: Must match pattern [A-Z]{5}[0-9]{4}[A-Z]{1}
- aadhaarNumber: Must be exactly 12 digits
- ifscCode: Must match pattern [A-Z]{4}0[A-Z0-9]{6}
- phone: Must be exactly 10 digits
- email: Must be valid email format
- creditScore: Must be between 300 and 900
- tenureMonths: Must be between 6 and 360
- loanAmount: Must be at least 10000
- employmentType: Must be "salaried" or "self-employed"
- loanType: Must be one of: "home", "gold", "personal", "vehicle", "education", "business"
- maritalStatus: Must be one of: "Single", "Married", "Divorced", "Widowed"
- residenceType: Must be one of: "owned", "rented", "parental", "company_provided"
- paymentHistory: Must be one of: "excellent", "good", "fair", "poor"
